청정수소 발전 역산 비교 - 어떻게 계산된 건지 설명
====================================================

[전제]
- 수소 연료전지가 24시간 × 365일 멈추지 않고 돌아간다고 가정
- 1년 = 365일 (윤년 보정 안 함)
- 필요 용량은 100kW(=0.1MW) 단위로 올림 (수소 연료전지가 100kW 모듈 단위로 팔리니까)


[화면에서 가져온 값 / 설정값]
- 한전 단가:     210원/kWh        (설정 → 한전 판매 단가)
- 수소 1MW 단가: 60억원           (설정 → 수소발전 단가)
- 12개월 사용량 합: 3,818,892 kWh (좌측 입력 패널 Sheet 4 사용량 합계)
- 12개월 전기료 합: 846,936,397원 ≈ 8.47억  (Sheet 5 전기요금 합계)


[하나씩 어떻게 나온 건지]

1) 연간 전기료 = 8.47억
   → Sheet 5의 1~12월 전기요금을 다 더한 값
   → 846,936,397원을 억으로 환산해서 표시

2) 연간 필요 발전량 = 3,818,892 kWh
   → Sheet 4의 1~12월 사용량(kWh)을 다 더한 값
   → 즉 "이 회사가 1년에 진짜로 쓴 전기량"
   → 사용량이 입력 안 됐을 때만 전기료÷한전단가로 폴백

3) 일일 필요 발전량 ≈ 10,463 kWh/일
   → 3,818,892 ÷ 365 ≈ 10,462.7
   → 그냥 365일로 나눈 거

4) 단순역산 (참고용) ≈ 4,033,030 kWh
   → 846,936,397 ÷ 210 = 4,033,030
   → "전기료를 한전단가로 나누면 몇 kWh가 나올까" 그냥 참고로 보여주는 것
   → 실측(3,818,892)이랑 약 5.6% 차이 나는 건 전기료에 기본료/계약가산금 등이 섞여있기 때문

5) 시간당 실측용량 ≈ 435.9 kW
   → 3,818,892 ÷ (24 × 365) = 3,818,892 ÷ 8,760 ≈ 435.9
   → 1년치 사용량을 24시간 × 365일로 나눠서 "평균적으로 몇 kW씩 써야 충당되나" 계산
   → 이게 진짜 평균 필요 출력

6) 필요 용량 = 0.50 MW (= 500 kW)
   → 435.9 kW를 MW로 바꾸면 0.4359 MW
   → 100kW 단위로 "올림" → 0.5 MW
     (0.4359 × 10 = 4.359 → 올림하면 5 → ÷10 = 0.5)
   → 즉 "500kW급 수소 연료전지가 있어야 충당된다"

7) 예상 투자비 = 30.00억
   → 0.5 MW × 60억/MW = 30억
   → 1MW당 60억이라는 단가에 필요 용량(0.5MW)을 곱한 것

8) ROI ≈ 3.54년
   → 30억 ÷ 8.47억 ≈ 3.54
   → "투자비 30억을, 매년 절감되는 전기료 8.47억으로 회수하면 3.54년 걸린다"는 의미


[흐름 한 줄로]
사용량 3,818,892 kWh
  ÷ (24 × 365 = 8,760)  → 435.9 kW
  ÷ 1000               → 0.4359 MW
  100kW 단위로 올림     → 0.50 MW
  × 60억               → 30억 (투자비)
  ÷ 8.47억 (연간 전기료) → 3.54년 (ROI)


[참고]
- 수소발전 매출 아니라 "전기료가 그만큼 안 나가니까 그걸로 회수"라는 개념
- 연료비, 가동률(95% 등), 유지보수비 등은 안 들어가 있음 (단순 비교용)
- 평균 필요 출력이 100kW 미만이면 자동으로 "태양광 PV 권장" 경고 뜸
- 한전단가/수소단가는 설정에서 바꾸면 즉시 다시 계산됨
